Api de Cadastro

Finalidade:

Ela servirá para cadastrar usuários, produtos e pedidos.
Isso será feito para facilitar o gerenciamento de um sistema de produtos facilitando a vida dos usuários.

Funcionalidade:

Cadastro de usuários, cadastro de produtos, registro de pedidos, listagem de dados, atualização de registros.

Tecnologias:

-CSS
-Node.js
-Express

Como rodar:

- npm install
- node index.js
